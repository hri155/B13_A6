# ResumeMart

ResumeMart is a modern and responsive product landing page built with React.js, Tailwind CSS, and React-Toastify. It showcases resume-related digital products in a clean marketplace layout with interactive cart functionality, responsive sections, and smooth user feedback.

# Description

This project is designed as a stylish frontend marketplace interface where users can browse resume products, add them to a cart, remove selected items, and proceed to checkout. It includes a polished hero banner, statistics section, product cards, pricing section, step-by-step guide, and footer.

# Technologies Used

- React.js
- Tailwind CSS
- DaisyUI
- JavaScript (ES6+)
- React-Toastify
- JSON data

# Features

- Responsive landing page with banner, stats, steps, pricing, and footer
- Product and cart toggle system with dynamic navbar cart count
- Add to cart, remove item, and proceed to checkout with toast alerts

# Optional Features Included

- Clicking 'Buy Now' increases the cart count in the navbar
- Clicking 'Proceed to Checkout' clears all cart items
- Product button text changes to 'Added to cart' after selection

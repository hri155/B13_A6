# ResumeMart

This project is a frontend demo marketplace application built for learning and practice purposes only.

The website demonstrates:
 React component structure
 Dynamic product rendering
 Cart functionality
 Responsive UI design
 Routing and state management

# Disclaimer

This is NOT a real commercial website.
No real payments, transactions, account creation, or checkout processing are implemented.All products, pricing, and features shown are fictional and used only for educational demonstration purposes.

# Description

This project is designed as a stylish frontend marketplace interface where users can browse resume products, add them to a cart, remove selected items, and proceed to checkout. It includes a polished hero banner, statistics section, product cards, pricing section, step-by-step guide, and footer. this fontend design is created for only educational purpose

# Technologies Used

- React.js
- Tailwind CSS
- DaisyUI
- JavaScript (ES6+)
- React-Toastify
- JSON data

# Features

- Responsive landing page with banner, stats, steps, pricing, and footer
- Product and cart toggle system with dynamic navbar cart count
- Add to cart, remove item, and proceed to checkout with toast alerts

# Optional Features Included

- Clicking Buy Now button increases the cart count in the navbar
- Clicking Proceed to Checkout button clears all cart items
- Product button text changes to Added to cart after selection
